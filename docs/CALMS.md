# Cultura CALMS en este Proyecto

- **C**ulture (Cultura): Fomentamos la colaboración entre desarrollo y operaciones mediante la automatización.
- **A**utomation (Automatización): Pipeline CI/CD automatiza pruebas, linting y seguridad.
- **L**ean (Lean): Procesos ligeros, infraestructura como código (IaC).
- **M**easurement (Medición): Logs estructurados, métricas (Prometheus) para visibilidad.
- **S**haring (Compartir): Repositorio centralizado, documentación clara.
