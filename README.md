# Proyecto Final DevOps - To-Do API

Este proyecto implementa una API robusta con pruebas unitarias, linter PEP 8, 
arquitectura de contenedores Docker multi-stage y un pipeline automatizado de GitHub Actions.

## Cómo ejecutar en local:
1. Asegúrate de tener Docker instalado.
2. Ejecuta: `docker compose up -d`
3. La API estará disponible en: `http://localhost:5000/`
